PetCare.

PetCare es un proyecto que ayuda a los dueños de mascotas a encontrar veterinarios y organizar pedir citas con ellos en caso de necesitarlas.

Funciones implementadas:
- Inicio de sesión con google, con github y en la propia web.
- Inicio de sesión como invitado, como usuario y como administrador, con diferentes vistas y acciones según el rol.
- Editar el perfil de usuario desde el dashboard.
- Posibilidad de introducir los datos de tus mascotas y editarlos.
- Exportación de mascotas para los administradores.
- Mapa para buscar veterinarios cercanos.
- Estadísticas visibles solo para los administradores.
- Filtración de mascotas según el animal, nombre, dueño.

Apis implementadas:
- DogAPI / CatAPI: Para poner fotos a las mascotas.
- OpenStreetMap: Mapa para encontrar veterinarios en tu zona.
- Api mail, intentado con mailjett y mailtrap. ( Nada :( ) Tras algunas actualizaciones dejó de funcionar y no pudo resolverse.
